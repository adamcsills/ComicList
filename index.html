Weekly Comic List

Download this project as a .zip file Download this project as a tar.gz file

This project reads the new comic feeds from www.comiclist.com, allows you to store the comics on your pull list, and shows you on a per-week basis what comics you should expect. It also lets you browse available comics on a weekly basis, using historical lists.

You can download and run the application from here: https://raw.githubusercontent.com/adamcsills/ComicList/master/ClickOnce/setup.exe

Note that Windows may catch the file with the Smart Screen Filter. This is because the file is brand new and there are only a few users currently.
